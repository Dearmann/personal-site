AngryBlocks

Game about throwing objects to knock down all the hostile objects despite obstacles of different strength levels. There are 3 types of throwables available to the player: regular, cluster and bombing. The flight trajectory of the thrown object is shown with dots. Destroying obstacles, destroying enemy blocks and not using all throwing objects gives the player additional points.

Chess

The aim of the project is to write an object oriented program in C++ for playing chess by two human players. Graphics are implemented using the SFML library. The program uses a two-dimensional array, which is very well suited to represent a chessboard. Chess pieces inherit from the abstract class Pieces along with a polymorphic function for making moves.
